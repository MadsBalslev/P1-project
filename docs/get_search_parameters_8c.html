Get the Dates object.

Først spørger vi om hvilken data arrengementet starter på, og reducere måneden med 1, da tm_mon i tm structen, tæller med måneder siden januar. Derefter ser vi om det er et valid input der er angivet, hvis det er det, så spørger vi om slutningen på arrengementet, og gør de samme ting som nævnt før. Til sidst printer vi inputtet vi har fået, og fastsætter start- og slutdato.
